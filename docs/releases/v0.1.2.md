## English

RustPad v0.1.2 focuses on editor usability: visual code folding, reliable syntax highlighting, and a complete color-mark workflow.

### Highlights

#### Code folding (new UI)

- Fold gutter between line numbers and code, similar to Notepad++ / VS Code.
- Yellow square icons (`−` expanded / `+` collapsed) on block start lines.
- Vertical scope lines show block extent; click an icon to fold or unfold.
- Auto-detects `enum`, `struct`, `fn`, `impl`, and other `{ ... }` blocks.
- Ignores braces inside strings and comments; supports indent-based folds for Python-like files.

#### Color marks

- Right-click color marks apply as **background** highlights (syntax colors unchanged).
- Gutter stripes align with marked lines, including multi-line selections.
- Marks follow text through copy, cut, paste, and keyboard edits.

#### Syntax highlighting

- Fixed cached highlighting skipping syntect state updates, which caused wrong colors on functions and keywords below the first cached lines.

#### Context menu & editing

- Copy/cut/paste and fold actions work reliably from the right-click menu.
- Selection snapshot for menu operations; cursor moves to right-click position for line-based actions.

### Install & build

```bash
git clone https://github.com/joyoki/rustpad.git
cd rustpad
git checkout v0.1.2
cargo build --release
cargo run --release
```

macOS app bundle (optional):

```bash
./scripts/build-macos.sh
```

## 中文

RustPad v0.1.2 聚焦编辑器体验：可视化代码折叠、稳定的语法高亮，以及完整的颜色标记工作流。

### 亮点

#### 代码折叠（新 UI）

- 行号与代码区之间新增折叠栏，风格类似 Notepad++ / VS Code。
- 块起始行显示黄色方块图标：`−` 表示展开，`+` 表示收起。
- 竖向作用域线标示代码块范围；点击图标即可折叠或展开。
- 自动识别 `enum`、`struct`、`fn`、`impl` 等 `{ ... }` 代码块。
- 忽略字符串与注释中的括号；支持 Python 等语言的缩进折叠。

#### 颜色标记

- 右键颜色标记作用于**文字背景**（不改变语法前景色）。
- 行号区色条与标记行对齐，支持多行选区。
- 复制、剪切、粘贴及键盘编辑时，标记随文本同步更新。

#### 语法高亮

- 修复行缓存未推进 syntect 解析状态，导致后续行函数名/关键字着色错误的问题。

#### 右键菜单与编辑

- 右键菜单中的复制/剪切/粘贴与折叠操作稳定可用。
- 菜单操作时快照选区；右键将光标移到点击处，便于按行折叠等操作。
