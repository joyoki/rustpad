## English

RustPad v0.1.3 improves navigation, encoding workflows, and macOS integration—bringing Notepad++-style scrolling and silhouettes alongside a full native menu bar on Apple Silicon MacBooks.

### Highlights

#### Quick scroll bar & document silhouette

- Narrow quick scroll strip with up/down arrows, draggable thumb, track click, wheel scroll, and context menu (scroll to cursor, top/bottom, page up/down).
- Inline gray line silhouette beside the scrollbar maps the full file; a viewport band follows scroll position with click/drag to jump.
- Scrollbar controls are flush-aligned (no gap between arrows, thumb, and track).

#### Encoding & settings

- Menu bar **Encoding** submenu: open with encoding, convert, and batch convert (ANSI/GBK, UTF-8, UTF-8 BOM, UTF-16, and more).
- Active **Open with encoding** choice shows a checkmark (✅) for at-a-glance feedback.
- Auto-detection defaults to **UTF-8** when no BOM is present.
- **Settings** menu with Preferences and Keyboard Shortcuts editor (Notepad++ / VS Code presets, rebind keys, conflict detection).
- Toolbar **encoding** button (🌐) restored with the same actions as the menu.

#### macOS native menu bar

- Full system menu bar via `muda`, with parity to Windows/Linux (File, Edit, View, Encoding, Tools, Settings, Help).
- Integrates with the Apple Silicon notch / menu bar safe area.

#### Editor UX

- **Column selection and copy**: Alt+drag rectangular selection, column copy via menu/shortcut.
- **Content extent line**: orange vertical guide on the line-number gutter (Notepad++ style).
- Toolbar icon-only buttons with hover tooltips; direct font-size input (8–72 px).
- Default editor font **16px**; line numbers **14px**; fold gutter left of line numbers.

#### Fixes

- Scroll wheel no longer jitters: auto-scroll-to-cursor runs only when the cursor moves.
- Save/save-as writes using the tab's selected encoding profile.

## 中文

RustPad v0.1.3 强化导航、编码工作流与 macOS 集成——带来 Notepad++ 风格的快速滚动条与文档剪影，并在 Apple Silicon MacBook 上提供完整原生菜单栏。

### 亮点

#### 快速滚动条与文档剪影

- 窄条快速滚动轨道：上下箭头、可拖滑块、点击轨道跳转、滚轮滚动及右键菜单（滚动到光标、顶/底、翻页等）。
- 滚动条旁内联灰色行条剪影映射全文；蓝色视口带随滚动移动，支持点击/拖拽跳转。
- 滚动条控件无缝贴合（箭头、滑块与轨道之间无间隙）。

#### 编码与设置

- 菜单栏 **编码** 子菜单：使用指定编码打开、转换、批量转换（ANSI/GBK、UTF-8、UTF-8 BOM、UTF-16 等）。
- **使用编码打开** 当前项前显示 ✅ 勾选，所见即所得。
- 无 BOM 时自动检测默认 **UTF-8**。
- **设置** 菜单：首选项与快捷键管理（Notepad++ / VS Code 方案、自定义按键、冲突检测）。
- 工具栏 **编码** 按钮（🌐）已恢复，与菜单栏编码子菜单功能一致。

#### macOS 原生菜单栏

- 通过 `muda` 提供完整系统菜单栏，与 Windows/Linux 功能对齐（文件、编辑、视图、编码、工具、设置、帮助）。
- 适配 Apple Silicon 刘海屏 / 菜单栏安全区域。

#### 编辑器体验

- **列选择与列复制**：Alt+拖动矩形列选、菜单/快捷键列复制。
- **内容长度竖线**：行号区橙色竖线（Notepad++ 风格）。
- 工具栏仅图标按钮 + 悬停提示；可直接输入字号（8–72 px）。
- 默认编辑字号 **16px**；行号 **14px**；折叠栏位于行号左侧。

#### 修复

- 滚轮滚动不再抖动：仅在光标移动时自动滚到光标可见。
- 保存/另存为按标签页所选编码写入文件。
