## English

RustPad v0.1.4 sharpens everyday editing: Scintilla-style character selection, a taller toolbar, localized chrome, safer file I/O, and reliable macOS/Windows release packages.

### Highlights

#### Editor selection & copy

- **Per-character hit testing** via a Scintilla-inspired `line_layout` module: caret positions are precomputed and clicks snap to the nearest character boundary.
- **Drag selection** tracks the mouse anchor correctly across small movements and releases cleanly after extended drags (no sticky second-click selection).
- Copy and cut now respect exact character boundaries, matching Notepad++ / Scintilla behavior.

#### UI & localization

- **Sidebar and toolbar i18n**: file tree, tabs, and toolbar labels follow the active locale.
- **Responsive toolbar**: compact icon-only mode on narrow windows; icon + label on wider layouts.
- **Long path ellipsis** in the sidebar so deep directories stay readable.
- Toolbar raised to **30px** with **14px** labels; font-size field text is centered vertically and horizontally.

#### Tabs, search & files

- **Tab context menu**: close, close others, close all, copy path, reveal in folder, open terminal.
- **Search history** remembers recent queries.
- **Drag-and-drop** files onto the window to open them.
- Scroll fixes for smoother navigation during search and tab switches.

#### Encoding & I/O

- **GBK and Latin-1 detection** restored when no BOM is present (alongside UTF-8).
- **Atomic saves**: write to a temp file then rename, reducing risk of truncated files on crash.
- Save/save-as and diff HTML export now **surface write errors**; exported diff HTML is properly escaped.

#### Platform & packaging

- **Windows** release builds as a GUI app (no console window).
- **macOS** release pipeline reads version from `Cargo.toml`; ad-hoc codesign fixed so **DMG** packaging completes in CI.
- Shell helpers to **reveal files in the OS file manager** and **open a terminal** at the file location.
- Optimized logo and icon assets for sharper window/taskbar rendering.

#### Code quality

- Clippy clean across the codebase; UI call sites simplified with parameter structs.

## 中文

RustPad v0.1.4 聚焦日常编辑体验：Scintilla 风格逐字符选择、加高工具栏、界面本地化、更安全的文件写入，以及可靠的 macOS/Windows 发布包。

### 亮点

#### 编辑器选择与复制

- 借鉴 Scintilla 的 **`line_layout`** 模块实现**逐字符命中测试**：预计算字符位置，点击吸附到最近字符边界。
- **拖拽选区**在小范围移动时正确跟踪锚点，大范围拖拽松开后不会与第二次选择粘连。
- 复制/剪切按精确字符边界截取，行为对齐 Notepad++ / Scintilla。

#### 界面与本地化

- **侧栏与工具栏 i18n**：文件树、标签页、工具栏文案随当前语言切换。
- **响应式工具栏**：窄窗口仅显示图标，宽布局显示图标 + 文字。
- 侧栏**长路径省略**显示，深层目录仍可读。
- 工具栏高度 **30px**、标签 **14px**；字号输入框文字上下左右居中。

#### 标签页、搜索与文件

- **标签页右键菜单**：关闭、关闭其他、全部关闭、复制路径、在文件夹中显示、打开终端。
- **搜索历史**记录最近查询。
- **拖放文件**到窗口即可打开。
- 搜索与切换标签时的滚动体验优化。

#### 编码与 I/O

- 无 BOM 时恢复 **GBK / Latin-1** 检测（与 UTF-8 并存）。
- **原子写入**：先写临时文件再重命名，降低崩溃时文件截断风险。
- 保存/另存为与 Diff HTML 导出**显示写入错误**；导出 HTML 正确转义。

#### 平台与打包

- **Windows** 发布版以 GUI 程序构建（无控制台窗口）。
- **macOS** 发布流程从 `Cargo.toml` 读取版本；修复 ad-hoc 签名，CI 可正常产出 **DMG**。
- 提供在系统文件管理器中**显示文件**、在文件位置**打开终端**的脚本辅助。
- 优化 logo 与图标资源，窗口/任务栏显示更清晰。

#### 代码质量

- 全项目通过 Clippy；UI 调用点用参数结构体简化。
