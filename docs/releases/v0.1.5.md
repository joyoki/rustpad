## English

RustPad v0.1.5 adds a full **Notepad-- style compare experience** in detached windows: file, folder, and binary comparison with synchronized side-by-side editing, diff navigation, and encoding controls.

### Highlights

#### Compare windows (Notepad-- style)

- **Detached compare viewports** for file, folder, and binary comparison — no longer embedded in the main editor window.
- **Unified compare chrome**: icon toolbar, left/right path bars with open/save actions, encoding row, and single-line diff statistics footer.
- **Side-by-side text diff** with line numbers, inline character highlights, merge arrows (▶/◀), and optional diff map strip.
- **Virtualized scrolling** for large files; click-to-edit individual lines with undo support.
- **Drag-and-drop** files or folders onto compare windows.

#### Diff engine

- **Binary comparison** for byte-level diffs with hex-style navigation.
- **Folder sync** utilities and enhanced **folder diff** (deep content mode, filters).
- Algorithms: **Myers**, **Patience**, **LCS**; strict mode and ignore-whitespace/case options.

#### Application integration

- **Compare menu** in the main window and **macOS menu bar** entries.
- Recent compare paths persisted in config.
- Full **i18n** (English / 中文) for compare UI strings.

## 中文

RustPad v0.1.5 带来完整的 **Notepad-- 风格对比体验**：独立弹出窗口支持文件、文件夹、二进制对比，左右同步编辑、差异导航与编码控制。

### 亮点

#### 对比窗口（Notepad-- 风格）

- **独立对比视口**：文件 / 文件夹 / 二进制对比，不再挤在主编辑窗口内。
- **统一界面**：图标工具栏、左右路径栏（打开/保存）、编码行、单行对比统计底栏。
- **左右并排 diff**：行号、字符级高亮、合并按钮（▶/◀）、可选差异图条。
- **大文件虚拟滚动**；点击行内编辑，支持撤销。
- 支持将**文件或文件夹拖入**对比窗口。

#### 对比引擎

- **二进制对比**（字节级差异与导航）。
- **文件夹同步**与增强的**目录对比**（深度内容模式、过滤）。
- 算法：**Myers**、**Patience**、**LCS**；严格模式、忽略空白/大小写。

#### 应用集成

- 主窗口 **对比菜单** 与 **macOS 菜单栏** 入口。
- 最近对比路径写入配置。
- 对比界面完整 **中英文 i18n**。
